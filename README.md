yii2-plupload
=============

Yii2 Plupload extension, provides useful widgets.

WIP...

Installation
------------
Install package via composer : `composer require soju/yii2-plupload:dev-master`

Usage
-----
```php
<?= \soju\yii2plupload\widgets\UIWidget::widget([
	'settings'=>[
		'url'=>['/controller/action'],
	],
]); ?>
```
